Assembly Process

Follow these steps to assemble your computer:

  1. Install the CPU on the motherboard.
  2. Attach the RAM to the motherboard.
  3. Mount the motherboard in the case.
  4. Connect the power supply.
  5. Install the storage drives.
  6. Install the graphics card.
